[White Codex] Devlog #1 - Magic and Dungeon

New Game that I am working on

I am excited to share that I am currently working on a new game that is inspired by some of the mechanics in “Survival Project”, which is game that I used to play when I was alot younger. “Survival Project” was a MMORPG with simple mechanics, but unfortunately, the game servers closed down in 2013. You can still play at a private server - Survival Project - Generations though, but the feel is just different…

That being said, I am not creating a MMORPG game, but instead picking out the components which I love about the game.

Brainstorming / Game Concepts

While brainstorming of a simple game with a small scope which I can work on as a solo indie game developer, I remembered this game called “Survival Project” which I used to spend quite a large amount of time on. And since the game was no longer available, I wanted to create a game with similar control mechanics for old times sake.

Hence, using “Survival Project” movement mechanics as a base, I am thinking of a game where you will play as a witch who explore dungeons and learn spells along the way. So, simply throwing ideas out now… here are some of the key elements which I will want to incorporate to my game.

1 - Top Down View

As someone who gets motion sickness easily, I prefer top down games compared to first-person or third-person games. Hence, games like brawl stars, death door, tunic, survival project, etc… appeals to me more. So for this new game, it will be a top down game.

2 - Movement Mechanics

I really like “old” movement mechanics in “Survival Project”. This form of movement mechanics basically moves your character forward like you would with a Tank, without any mouse to control the directions. I don’t think you can find this in modern games easily, hence, this will be something I would want to replicate.

3 - Magic Circles

Lastly, what is casting spells without maagic circles! Whether it’s games like “Surival Project” or animes like “Fairy Tail”. I like the idea of using magic circles for magic effects. So this might be something I would like to try. Hopefully, it ain’t too difficult…

Summary

Anyway, that’s it for now! Look forward to more updates as I develop this game :)
