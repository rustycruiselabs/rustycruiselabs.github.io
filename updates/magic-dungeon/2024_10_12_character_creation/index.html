[White Codex] Devlog #2 - Designing Main Character

This post is to document on my character creation process. This is something that I am not very good with and am really learning along the way as it is my first time creating 3D characters. The tools I am using are krita for sketching my art, blender for modeling & animations and unity for game physics.

Sketching

After researching on a couple of videos, I realized that it is important to have a sketch of the character before I start modelling. So here’s a rough sketch that I drew using krita. The main concepts are the extremely large witch hat and student-like attire which was inspired from characters in Wistoria: Wand and Sword. For reference, I think you can check out these videos which I found useful in helping me get started with the sketch

  • Keven - How I learned to Create Retro Game Characters
  • Solomon gumball - How a comic book artist gave my game personality - Devlog 7
  • Jelle Vermandere - How to Make a 3D Character For Your Game (Blender to Unity)
  • Iacruzo - Creating Stylized Low Poly Characters in Blender
